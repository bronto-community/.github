# .github

This is the special `.github` repository for the [bronto-community](https://github.com/bronto-community) organization. It holds:

- [`profile/README.md`](profile/README.md) — the org profile shown on [github.com/bronto-community](https://github.com/bronto-community)
- [`CONTRIBUTING.md`](CONTRIBUTING.md) — default contributing guidelines, inherited by every repo in the org that doesn't define its own

Changes to `profile/README.md` go live on the org page as soon as they're merged to `main`.
